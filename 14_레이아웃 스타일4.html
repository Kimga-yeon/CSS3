배치 관련 스타일(position)

요소의 위치를 지정하는 속성

position : relative / absolute 

            relative : 해당 속성이 적용된 요소가 아닌 요소 내 하위 요소들을 left, right, top, bottom 속성을 이용하여 위치를 지정할 수 있다.
            absolute : 요소의 기본 배치 순서를 무시하고 지정된 절대 위치에 요소를 표시함
                        단, 상위요소가 relatve, absolulte, fixed인 경우 상위 요소를 기준으로 하여 상대적 위치를 가짐

position : fixed; (고정 위치)

고정 위치는 브라우저 창을 기준으로 요소가 항상 같은 위치를 가지게 하는 속성

요소 Stack 순서 지정 스타일(z-index)

요소가 쌓이는 순서를 지정하는 속성
지정된 숫자가 클수록 위쪽에 쌓임
(주의사항)
z-index는 relative, absolute, fixed position이 설정된 요소에서만 정상 동작함
-> z-index가 정상 동작하지 않을 경우 position : relative를 추가하면 됨

요소 표시 여부 지정 스타일(visibility)

페이지에 특정 요소를 보이거나, 안 보이게 하거나, 없애는 속성

요소 정렬 스타일(float/clear)

            float: 요소를 띄워서 정렬하는 속성
                   요소들이 float로 설정된 방향으로 떠서 흐르듯이 정렬되어 배치됨
            clear: float로 인해 요소들이 띄워져 있는 상태를 해제하는 속성
                   해제할 때 float의 특정 방향을 지정하여 해제 가능(left, right, both)
