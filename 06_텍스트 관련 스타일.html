텍스트 관련 스타일

color 속성 : 텍스트의 색상을 지정할 수 있다.

            선택자 { color : 색상명영문이름 | 16진수숫자 | rgb값 | rgba값 | hsl값 | hsla값; }
  • 색상명으로 지정
  • 16 값으로 지정
  • rgb 값으로 지정
  • rgba 값으로 지정
  • hsl 값으로 지정
  • hsla 값으로 지정

text-decoration 속성 : 텍스트에 줄을 긋거나 없애는 속성

  • 텍스트 영역 아래에 줄 긋기
  • 텍스트 영역 중간에 줄 긋기
  • 텍스트 영역 위에 줄 긋기
  • 하이퍼링크 밑줄 없애기

text-transsform 속성 : 영문 텍스트의 대소문자 변환 시 사용

            선택자 { text-transform : none | capitalize | uppercase | lowercase; }
  • capitalize : 첫 글자만 대문자로 변형
  • uppercase : 모든 영문자를 대문자로 변형
  • LOWERCASE : 모든 영문자를 소문자로 변형

text-shadow 속성 : 텍스트에 그림자 효과를 주는 속성

            선택자 { text-shadow : none | 가로거리 세로거리 번짐정도 색상; }
