자손, 후손 선택자

자손 선택자 : 선택된 태그 바로 하위 요소
후손 선택자 : 선택된 태그 하위에 있는 모든 요소 

            자손 선택자 : 선택자1>선택자2 { 설정 내용; }
                                    ㄴ 자식 선택자
            후손 선택자 : 선택자1 선택자2 {설정 내용; }
                                    ㄴ 후손 선택자(선택자 1 하위에 있는 모든 선택자2)

동위 선택자

동위 관계에서 뒤에 위치한 태그를 선택할 때 사용

            바로 뒤의 요소를 선택할 때 : 선택자1+선택자2 {설정내용;}
            뒤에 있는 모든 요소 선택 : 선택자1~선택자2{설정 내용;}

반응 선택자

사용자의 움직임에 따라 달라지는 선택자

            선택자 : active{ 설정내용; } 
            -> 사용자가 해당 선택자를 클릭했을 때 설정 내용으로 스타일이 변함
            선택자 : hover{ 설정내용; } 
            -> 사용자가 해당 선택자에 마우스를 올려놓았을 때 설정 내용으로 스타일이 변함

입력 양식에 따른 선택자(상태 선택자)

            checkbox 또는 radio input 태그가 선택이 되었을 경우 스타일을 변경할 때 사용함

            input태그 선택자:checked {설정 내용;}
