글꼴 관련 스타일

font-family 속성 : 텍스트의 글꼴을 지정하는 CSS 속성

            선택자{font-familt:글꼴1 이름, 글꼴2 이름, 글꼴3 이름; }
            글꼴1이 해당 장치에 설치되지 않았다면 글꼴2로 대체. 2도 없으면 3으로 대체. 3도 없으면 기본값 사용

font-size 속성 : 텍스트 크기 변경

            선택자{ font-size:크기;}
            *고정/가변 크기가 존재함
  • 고정크기 px 테스트
  • 가변크기 % 테스트
  • 가변크기 em 테스트

font-weight 속성 : 텍스트 두께를 표현할 때 사용하는 속성

            선택자{font-weight:normal | bold | bolder | lighter | 100~900; }
            
            bold : 두껍게(진하게)
            bolder : 상위 요소에서 상속된 굵기보다 더 두껍게(진하게)
  • 그냥 굵게
  • 얇게
  • font-weight:500

    상위보다 더 굵게

font-style 속성 : 텍스트 기울기 지정

                선택자 { font-style : normal | italic | oblique; }

italic 기울임 글꼴

oblique 기울임 글꼴

font 속성 : 글꼴 관련 스타일을 한 번에 지정 가능

                선택자 { font : font-style font-variant font-weight font-size/line-height font-family; }
