영역 관련 속성

1) content : 태그에 작성된 내용이 표시되는 영역
2) border : 요소 테두리 영역
3) padding : content와 border 사이 영역
4) margin : 타 요소와의 간격을 지정하는 영역
현재 div 전체 가로 길이
= width + 양쪽 padding + 양쪽 border + 양쪽 margin
= 500 + (20 * 2) + (20 * 2) + (20 * 2)
= 420px
현재 div 전체 세로 길이
= heigt + 상하 padding + 상하 border + 상하 margin
= 200 + (20 * 2) + (20 * 2) + (20 * 2)
= 320px
