화면 배치 방법 변경

block 요소, inline 요소의 속성을 변경하여 배치 방식을 변경할 수 있다.

block 요소를 inline 요소로 변경

            display : inline;
            지정된 요소의 형식이 inline으로 변경되어 화면을 수평분할하고 width, height 속성이 무시된다.

inline 요소를 block 요소로 변경

            display : blcok;
            지정된 요소의 형식이 blcok으로 변경되어 화면을 수평분할하고 width, height 속성이 적용된다.

inline-block 요소로 변경

            display : inline-block;
            영역 분할 방법은 inline 요소의 방법인 수평 분할로 지정되지만
            내용(content) 작성 부분은 block 요소로 지정된다. width, height 속성 지정이 가능하다.
